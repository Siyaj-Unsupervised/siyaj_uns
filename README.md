# siyaj_uns
### وصف البيانات (Datasets Description)
* **بيانات يوم الاثنين (Monday-WorkingHours):** تمثل حركة الشبكة الطبيعية (Normal Traffic) وتُستخدم لتعريف النظام بالسلوك السليم (Baseline).
* **بيانات يوم الأربعاء (Wednesday-WorkingHours):** تحتوي على حركة شبكة تتضمن هجمات حقيقية (مثل DoS و Slowloris) وتُستخدم لاختبار قوة النظام في اكتشاف التهديدات.
